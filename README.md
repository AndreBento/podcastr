## 🚀 Getting started

Clone o projeto e acesse a pasta

```bash
$ git clone https://github.com/AndreBento/podcastr.git && cd podcastr
```

Siga os passos abaixo
```bash
# Install the dependencies
$ yarn

# Run the JSON server
$ yarn server

# Start the project
$ yarn dev
```
